# Security Policy

## Supported Versions

Intensicare currently supports security fixes on:

- `main` (latest stable)

## Reporting a Vulnerability

Please report vulnerabilities through GitHub private reporting:

- https://github.com/Omni-Saude/intensicare/security/advisories/new

If private reporting is unavailable, contact the maintainers directly and avoid public disclosure until a fix is released.

## Response Targets

- Initial triage: up to 3 business days
- Status update: up to 7 business days
- Fix or mitigation timeline: based on severity and exploitability

## Scope

Security reports are in scope for:

- API/backend (`src/`)
- Frontend (`frontend/`)
- CI/CD workflows (`.github/workflows/`)
- Container definitions (`Dockerfile`, `frontend/Dockerfile`)
